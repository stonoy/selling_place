# selling_place
Selling Place — An online marketplace app where users can list items for sale and browse listings in a seamless, intuitive interface. Users sign up or log in with their Google account for streamlined access, and payments are securely processed via Razorpay. The app supports features like listing items, editing/deleting listings, browsing categories, search & filter, and integrated payment for transactions.

Live : https://selling-place.onrender.com
